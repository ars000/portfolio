Обо мне

Приветствую Вас на своем сайте-портфолио. Меня зовут Арсен. Занимаюсь HTML-версткой сайтов и приложений. Верстка полностью валидная, кроссбраузерная и адаптивная. Для обеспечения выполнения требований, предъявляемых к верстке сайта, приложения в процессе работы над проектом использую соответствуещие инструменты и технологии. Препроцессоры Sass/Less для ускоренной стилизации сайта и более гибкого кода, фреймворки Bootstrap/Material UI для кроссбраузерности и адаптивной верстки, сборщик проектов Gulp/Webpack для автоматизации процесса верстки, Pixel Perfect для совпадения верстки с предоставленным макетом «пиксель в пиксель». К сайту так же, при желании подключаются и настраиваются различные JS/jQuery библиотеки и плагины. Свёрстаный код проверяется W3C валидатором на соответствие стандартам. В конце работы осуществляется проверка отображения верстки в различных популярных браузерах на разных устройствах.
